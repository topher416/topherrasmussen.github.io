# For Developers

Technical guide for using the socrates-source corpus in AI/ML pipelines, applications, and research.

## Corpus Stats

The corpus contains 33 texts totaling approximately:
- ~2.8M characters of Greek text
- ~470K words
- ~1.1M tokens (Claude/GPT-4 tokenizers)
- ~3MB as a single concatenated file

The entire corpus fits within a single context window of Claude (200K tokens) or Gemini (1M+ tokens). For GPT-4 (128K tokens), use a subset or RAG approach.

## Token Counts

Run `python scripts/count_tokens.py` for exact counts. Approximate token estimates per text:

| Text | Est. Tokens | Notes |
|------|-------------|-------|
| Republic | ~220K | Largest single text (10 books) |
| Memorabilia | ~90K | Xenophon's largest |
| Gorgias | ~67K | Long dialogue |
| Timaeus | ~60K | Cosmological monologue |
| All Plato | ~900K | 28 dialogues |
| All Xenophon | ~170K | 4 works |
| Aristophanes | ~27K | The Clouds only |

Greek text tokenizes at roughly **2.3–2.8 characters per token** depending on the tokenizer, due to multi-byte UTF-8 encoding of polytonic Greek.

## Embedding Strategies

### Chunking by Stephanus Page

Each text is divided into Stephanus pages (for Plato) or numbered sections (for Xenophon/Aristophanes), marked with `### ` headers. This gives natural semantic chunks of roughly 200–500 tokens each.

```python
import re

def chunk_by_section(text_md):
    """Split a text.md file into chunks by section headers."""
    sections = re.split(r'\n### ', text_md)
    chunks = []
    for section in sections[1:]:  # Skip header
        lines = section.split('\n', 1)
        section_id = lines[0].strip()
        content = lines[1].strip() if len(lines) > 1 else ""
        if content:
            chunks.append({
                "section": section_id,
                "text": content
            })
    return chunks
```

### Chunking by Speaker Turn

For dialogues with speaker labels (e.g., `ΣΩ.` for Socrates, `ΚΡ.` for Crito), you can chunk by speaking turn:

```python
def chunk_by_speaker(text):
    """Split dialogue text by speaker turns."""
    # Common speaker abbreviations in the corpus
    pattern = r'(ΣΩ\.|ΚΡ\.|ΓΛ\.|ΑΔ\.|ΠΡΩ\.|ΓΟΡ\.|ΠΩΛ\.|ΚΑΛ\.)'
    turns = re.split(pattern, text)
    # Pair speaker labels with their text
    result = []
    for i in range(1, len(turns), 2):
        result.append({
            "speaker": turns[i].rstrip('.'),
            "text": turns[i+1].strip()
        })
    return result
```

### Recommended Embedding Models

- **text-embedding-3-large** (OpenAI) — 3072 dimensions, good multilingual support
- **voyage-multilingual-2** (Voyage AI) — Strong on non-Latin scripts
- **multilingual-e5-large** (open source) — Good baseline for Greek

Note: Most embedding models were not specifically trained on Ancient Greek. Test retrieval quality on known passages before committing to a model.

### Global Metadata

- `corpus.json` — Master index (generated by `build_corpus_json.py`)
- `metadata/characters.json` — All named characters across all texts
- `metadata/themes.json` — Thematic index with Greek terms
- `metadata/timeline.json` — Dramatic chronology
- `metadata/cross-references.json` — Inter-text connections
- `metadata/socrates-speaking.json` — Where and how much Socrates speaks

## Building a "Talk to Socrates" Bot

### Architecture

```
User query
    ↓
Embed query → Search corpus chunks → Retrieve top-k passages
    ↓
Construct prompt with:
  - System: "You are Socrates. Respond based only on the Greek source texts provided."
  - Retrieved passages (Greek + metadata context)
  - User's question
    ↓
LLM generates response grounded in sources
```

### Implementation with Claude

```python
import anthropic

client = anthropic.Anthropic()

# Load a text and its metadata
with open("texts/plato/apology/text.md") as f:
    greek_text = f.read()
with open("texts/plato/apology/metadata.json") as f:
    metadata = json.load(f)

response = client.messages.create(
    model="claude-sonnet-4-6",
    max_tokens=4096,
    system="""You are a scholarly assistant with access to the original Greek texts
of Plato's dialogues. When answering questions about Socrates, ground your
responses in the specific Greek passages provided. Translate relevant passages
and explain your translation choices.""",
    messages=[{
        "role": "user",
        "content": f"""Here is the Greek text of Plato's Apology:

{greek_text}

Metadata: {json.dumps(metadata, indent=2)}

Question: What does Socrates say about the examined life? Quote the Greek
and provide a close translation."""
    }]
)
```

### With Context Caching (Claude)

For repeated queries against the same corpus, use prompt caching to avoid re-processing the Greek text:

```python
# First request caches the corpus
response = client.messages.create(
    model="claude-sonnet-4-6",
    max_tokens=4096,
    system=[{
        "type": "text",
        "text": "You are a Socratic scholar with the complete Greek corpus.",
        "cache_control": {"type": "ephemeral"}
    }],
    messages=[{
        "role": "user",
        "content": [{
            "type": "text",
            "text": corpus_text,  # Full corpus
            "cache_control": {"type": "ephemeral"}
        }, {
            "type": "text",
            "text": "What does Socrates say about justice in the Republic?"
        }]
    }]
)
# Subsequent requests reuse the cached corpus at reduced cost
```

## Integration Examples

### LangChain

```python
from langchain.text_splitter import MarkdownHeaderTextSplitter
from langchain_community.vectorstores import Chroma
from langchain_openai import OpenAIEmbeddings

# Split by Stephanus sections
headers = [("###", "section")]
splitter = MarkdownHeaderTextSplitter(headers_to_split_on=headers)

docs = []
for text_path in glob("texts/**/text.md"):
    with open(text_path) as f:
        chunks = splitter.split_text(f.read())
        for chunk in chunks:
            chunk.metadata["source"] = text_path
            docs.append(chunk)

# Create vector store
vectorstore = Chroma.from_documents(docs, OpenAIEmbeddings())
```

### LlamaIndex

```python
from llama_index.core import SimpleDirectoryReader, VectorStoreIndex

# Load all texts
documents = SimpleDirectoryReader(
    "texts",
    recursive=True,
    required_exts=[".md"]
).load_data()

# Build index
index = VectorStoreIndex.from_documents(documents)
query_engine = index.as_query_engine()

response = query_engine.query(
    "What arguments does Socrates make about the immortality of the soul?"
)
```
